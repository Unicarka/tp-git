# Introduction

Ceci est une documentation simple. Vous devez compléter ce fichier dans un ticket.

Explication : Un paragraphe simple se compose de trois éléments principaux. La première phrase, souvent déclarative, est appelée « phrase thématique ». Elle introduit le sujet du paragraphe et en définit le ton. Les phrases suivantes développent, expliquent et illustrent le sujet abordé

- Git est un système de controle de version distribué

- un clone local du projet est un référentiel complet de controle de version

- Un logiciel VCS qui permets aux professionnels de la data de sauvegarder l'historique de modifications de leurs projets
